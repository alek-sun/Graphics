--Описание программы--
К ToolBar добавлены кнопки:
1. Save game - сохранить состояние поля в текстовый файл
2. Load game - загрузить состояние поля из файла
3. Run - запустить игру (шаги идут по таймеру 1 сек.)
4. Step - 1 шаг вперед
5. Clear - очистить состояние поля (все клетки становятся мертвыми)
6. Display impact - показать/скрыть значения impact у клеток
7. Replace/XOR - смена режима Replace/XOR
8. Size settings - настройка размеров поля и клеток
9. Game parametrs - настройка параметров игры
10. About author - об авторе
